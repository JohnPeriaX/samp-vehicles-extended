# Vehicles Extended

samp.saa patching based on:
[OpenSaa](https://gitlab.com/prime-hack/samp/plugins/OpenSaa/-/tree/master)
(Applied patch by looking patterns instead of hardcoded addresses, which works on all versions)


Vehicle ID limit (id > 400 | id < 611) was patched by pattern finding, so it should work on most versions.


## Supported Versions

- 0.3.7 R3
- 0.3.7 R4
- 0.3.7 R5
- 0.3.DL-R1
